Project Name: Re-AIM

Description:
Re-AIM is a multi-client Java chat application supporting rooms, private messaging, persistent chats, and Rock Paper Scissors.

Compilation:
javac *.java

Run Server:
java ServerMain

Run Client:
java ClientMain

Commands:
/list
/join roomName
/leave
/dm username message
/rps rock|paper|scissors

Features:
- Multi-client support
- Room-based chats
- Persistent room logs
- Private messaging
- Rock Paper Scissors
- Unique usernames
